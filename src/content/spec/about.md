# 暗涌层流 | Under-Laminar

## 从混沌中，看见脉络。

这是一个致力于 **系统思维** 与 **认知工程** 的数字实验室。我们相信，任何复杂的系统——无论是一款桌游、一个人工智能模型，还是一种社会现象——其内部都存在着两种根本动力：

*   **暗涌** :: 表面之下，混沌、交织、充满不确定性的湍流。
*   **层流** :: 其中清晰、有序、可被解析与应用的稳定脉络。

本博客通过持续的项目与写作，将这一理念付诸实践：

1.  **解码算法黑盒** -- 剖析前沿AI模型的运作逻辑，理解其内在秩序。
2.  **构建思考框架** -- 锻造分析复杂问题的通用思维工具。
